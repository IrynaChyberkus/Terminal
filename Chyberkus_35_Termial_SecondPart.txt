Termial. HW_2
 1. Сделать папку dir_1 - mkdir dir_1
 2. Зайти в папку dir_1 - cd dir_1
 3. Создать папку inner_dir_1 - mkdir inner_dir_1
 4. Посмотреть где ты находишься - pwd
 5. Находясь в папке dir_1 создать пустой текстовый файл tf_1.txt - touch tf_1.txt
 6. Находясь в папке dir_1 через команду cat создать текстовый файл tf_2.txt со следующими строками:
- the first 1
- the second 2
- the third 3 - cat > tf_2.txt переход на новую строку нажатием Enter окончание редактирование комбинацией Ctrl+C
 7. Зайти в папку inner_dir_1 - cd inner_dir_1
 8. Через cat сделать текстовый файл tf_3.txt  c любыми строками - cat > tf_3.txt и что-то записать, закончив ввод комбинацией Ctrl+C
 9. Через cat добавить в текстовый файл tf_3.txt строку “the second 2” - cat >> tf_3.txt затем внести the second 2 затем Enter затем Ctrl+C
 10. Через cat добавить в текстовый файл tf_3.txt строку “the sec 2” - cat >> tf_3.txt затем внести the sec 2 затем Enter затем Ctrl+C
 11. Через cat добавить в текстовый файл tf_2.txt строку “the sec 3” - вначале cd .. затем cat >> tf_2.txt затем внести the sec 3 затем Enter затем Ctrl+C
 12. Через cat добавить в текстовый файл tf_3.txt строку “the SeCoNd 2” - cat >> inner_dir_1/tf_3.txt затем внести the SeCoNd 2 затем Enter затем Ctrl+C
 13. Через cat добавить в текстовый файл tf_2.txt строку “the seConD 2” - cat >> tf_2.txt затем внести the SeCoNd 2 затем Enter затем Ctrl+C
 14. Сделать текстовый файл tf_4.txt в котором будет 15 строк. - vim tf_4.txt войти в режим редактирования клавишей I выполнить перевод каретки Enter 13 раз нажать Esc затем :wq
 15. Сделать текстовый файл tF_5.txt в котором будет 13 строк. - vim tf_5.txt войти в режим редактирования клавишей I выполнить перевод каретки Enter 11 раз нажать Esc затем :wq
 16. Вывести список всех файлов в папке. - ls -la или  find . (покажет все в текущей директории в том числе поддиректории с содержимым)
 17. Выйти из папки inner_dir_1 - cd .. (Если ты в ней находишься, так как она самая глубокая по вложенности здесь)
 18. Вывести содержимое файла tf_3.txt в терминал. - cat inner_dir_1/tf_3.txt
 19. Найти путь к файлу tf_4.txt - realpath tf_4.txt или  readlink -f tf_4.txt или find $(pwd) -name tf_4.txt
 20. Отчистить файл tf_4.txt от содержимого без удаления самого файла. - cat > tf_4.txt затем комбинацию Ctrl+C
 21. Найти путь к файлам у которых есть  “tf” в названии. -  find  -type f -name "*tf*"
 22. Найти путь к файлам у которых есть  “tf” в названии и буквы в любом регистре. - find -type f -iname "*tf*"
 23. Найти строки в файлах где есть комбинация букв “sec” в текущей папке - grep  sec ./* или grep -h sec ./* (если нужны только сами строки без пути)
 24. Найти строки в файлах где есть комбинация букв “sec” в любом регистре в текущей папке - grep -i sec ./* или grep -h -i sec ./* (если нужны только сами строки без пути)
 25. Найти строки в файлах где есть только комбинация букв “sec” в текущей папке - grep -w sec ./* или grep -h -w sec./* (если нужны только сами строки без пути)
 26. Найти строки в файлах где есть только комбинация букв “sec” в любом регистре в текущей папке - grep -i -w sec ./* или grep -h -i -w sec ./* (если нужны только сами строки без пути)
 27. Найти строки в файлах где есть комбинация букв “second” в текущей папке - grep second ./* или grep -h second ./* (если нужны только сами строки без пути)
 28. Найти строки в файлах где есть комбинация букв “second” в любом регистре в текущей папке - grep -i second ./* или  grep -h -i second ./* (если нужны только сами строки без пути)
 29. Найти строки в файлах где есть комбинация букв “second” во всех папках ниже уровнем - grep -r second или grep -h -r second (если нужны только сами строки без пути)
 30. Найти только путь и название файла в строках которых есть комбинация букв “second” в текущей папке -  grep -l second ./*
 31. Найти все строки во всех файлах где нет комбинации “second” -  grep -r -v second или grep -h -r -v second (если нужны только сами строки без пути)
 32. Найти только название и путь к файлам где нет комбинации “second” - grep -L -r  second
 33. Вывести в терминал 4 последних строк любого текстового файла - tail -4 inner_dir_1/tf_3.txt
 34. Вывести в терминал 4 первые строки любого текстового файла. - head -4 inner_dir_1/tf_3.txt
 35. Команда в одну строку. Создать папку и создать текстовый файл с содержиммым. -   mkdir inner_dir_2 && cat > inner_dir_2/tf_6.txt
 36. Команда в одну строку. Переместить в любую одну папку текстовые файлы у которых в содержимом есть слово “sec” - grep -rlw sec | xargs mv -t inner_dir_3/
 37. Команда в одну строку. Скопировать в любую одну папку текстовые файлы у которых в содержимом есть слово “sec” - grep -rlw sec | xargs cp -t inner_dir_2/
 38. Команда в одну строку. Найти все строки c “sec” во всех текстовых файлах, скопировать и вставить эти строки в один новый созданный текстовый файл. - grep -h -r -i sec . > sec.txt
 39. Команда в одну строку. Удалить текстовые файлы у которых в содержимом есть слово “sec” -  grep -rl sec | xargs rm
 40. Просто вывести в терминал строку “Good job!!” -  echo "Good job!"
